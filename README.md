# Clash of BaNG

Clash of BaNG is a cute fantasy fan-community website created for a Human and Computer Interaction project. The website introduces a playful community concept through character troops, gallery content, and a registration form.

## Project Description

This website was designed and developed as part of a Human and Computer Interaction assignment. The main goal of this project is to create a simple, interactive, and visually consistent website based on a cute fantasy theme.

The website was first designed using Figma, then implemented using HTML, external CSS, and JavaScript.

## Pages

The website consists of five main pages:

* Home Page
  Introduces Clash of BaNG with a hero section, short description, carousel, and most popular troop.

* About Page
  Explains the background, story, milestones, and creator information of Clash of BaNG.

* Troops Page
  Displays troop characters with images, short descriptions, and hover stats.

* Gallery Page
  Displays visual gallery content and an embedded YouTube video reference.

* Register Page
  Provides a registration form with JavaScript validation.

## Features

* Responsive layout
* Navigation between pages
* External CSS styling
* Carousel section
* Troop hover stats
* Register form validation
* Sound effect on submit button
* Simple page transition
* Sakura petal animation
* YouTube embed on Gallery Page
